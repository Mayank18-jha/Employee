# Employee Management System

## Problem Statement
Managing employee and department data manually is inefficient. This project automates employee records using SQL.

## Features
- Department & employee management
- SQL JOIN-based reports
- Persistent SQLite database

## Tech Stack
- Python
- SQLite
- SQL

## How to Run
python employee_system.py

## Future Improvements
- Update & delete employees
- Role-based access
- Web dashboard
